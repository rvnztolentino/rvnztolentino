# Renz Tolentino - Personal Website

A personal portfolio and showcase website for Renz Tolentino.

## Overview
- Minimalist, dark-themed design
- Integrated AI assistant (Kei) powered by Gemini
- Portfolio showcase
- Contact form

## Tech Stack
- React (Vite)
- TypeScript
- Tailwind CSS
- Google Gemini API

## Getting Started
1. `npm install`
2. Create `.env.local` with `VITE_GEMINI_API_KEY`
3. `npm run dev`
